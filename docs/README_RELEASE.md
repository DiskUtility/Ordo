# Ordo: Planner Release Notes (Concept)

## Version

`1.0 Concept`

## Highlights

- Local-first student planner foundation
- Today dashboard with deadline triage
- Timeline course management
- Task management with priority and status
- Reminder scheduling via local notifications
- Settings-based display preferences

## Quality Status

- Build verified via `xcodebuild`
- Unit and UI tests present in repo
- No cloud dependency

## Privacy

- No account required
- No third-party analytics/crash SDK
- Data stays on device

## Known Constraints

- Single active term
- No sync/import/export
- No iPad-specific custom layout pass yet
