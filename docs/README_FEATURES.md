# Ordo: Planner Features

## Student Setup

- Optional display name
- Student level profile (`High School` or `College`)
- Active term setup

## Timeline

- Add/edit/delete courses
- Day-of-week schedule metadata
- Time range and location support
- Compact card display mode support

## Tasks

- Add/edit/delete tasks
- Link tasks to courses
- Status (`Not Started`, `In Progress`, `Completed`)
- Priority (`Low`, `Medium`, `High`)
- Compact filter button with expandable controls

## Dashboard (Today)

- Greeting + student name
- 12h/24h clock toggle gesture
- Triage buckets:
  - Overdue
  - Due Today
  - Upcoming (next 7 days)

## Settings

- Student profile edits
- Reminder lead time
- Display controls:
  - Compact cards
  - Show/hide course codes
- Data utilities:
  - Clear completed tasks
  - Data summary
  - Debug mock data/reset (debug builds)
