# Ordo: Planner Product Brief

## Positioning

Ordo: Planner is a focused student planning app designed around one core promise: keep classes and deadlines clear every day.

## Target Users

- High school students
- College and university students

## Core Value

- Reduce deadline stress with simple triage
- Keep weekly class structure visible
- Stay local-first and private by default

## Scope (Current)

- Single active term model
- Course timeline CRUD
- Task CRUD with status and priority
- Dashboard triage buckets
- Local notification reminders

## Out of Scope (Current)

- Cloud sync
- Accounts and sign-in
- Third-party integrations
- AI features
- Widgets and import/export
